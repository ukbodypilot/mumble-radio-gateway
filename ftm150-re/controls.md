# FTM-150 Control Map

## Rotary Knobs (rotate + push)
- UL: Upper Left
- LL: Lower Left
- UR: Upper Right
- LR: Lower Right

## Buttons
- SDX
- BAND
- PMG
- VM
- F
- PWR

## Capture naming convention
- ul_cw    / ul_ccw    / ul_push
- ll_cw    / ll_ccw    / ll_push
- ur_cw    / ur_ccw    / ur_push
- lr_cw    / lr_ccw    / lr_push
- btn_sdx  / btn_band  / btn_pmg  / btn_vm  / btn_f  / btn_pwr
